      今年基金起什么名字会中呢？

      Post on 2014 / 09 / 26

      最近，今年的国家自然基金放榜了，周围有国内的同事中了，那契而不舍的精神着实让我感动，也让我看到了未来水深火热的科研之路的不平坦。

      听说，有时候给申请书起名字是个学问，有人只改了个名字来年就高中的故事也屡见不鲜。所以，我就动起了小聪明，希望能从今年获得资助的申请书的题目中寻找一些端倪。嘿嘿嘿，如果经过我的分析，搞到了最佳题目，我就能升职加薪、变成高富帅，心里想想还有点小高兴呢。

      因为我是搞生态的，只对生命科学类的感兴趣，所以我找到了今年的所有题目和获得资助的金额，利用Rwordseg包的默认分词算法做了一些小分析，得到如下表格。

      上表的左半边从上至下是按照单词的出现频率排序最高的前10个，右半边是按照平均每个单词所关联的项目金额排序的最高前10个。

      所以，基本上，我们要草拟的题目从这里面提取准没错。

      比如：

      • miR-II调控球菌稳定再生作用的分子机制
      • 氨基的基因地理作用及其在染色细胞编程中的功能研究

      哇！一看就显得非常高大上啊，好的题目是基金申请成功的关键，这么说，我马上就能升职加薪、傍上白富美、变成高富帅、走上人生巅峰了吗？
